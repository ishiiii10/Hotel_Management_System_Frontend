# Images Directory

## Hero Background Image

Place your luxury hotel hero background image here with the filename:
- `hero-background.jpg` (or `.png`)

The image should be:
- High resolution (at least 1920x1080px recommended)
- Optimized for web (compressed JPG format)
- Luxury hotel aesthetic (outdoor patio, pool area, or elegant hotel exterior)

If you don't have a local image, the application will use a fallback image from Unsplash.
